# Dataset

The dataset used in this project is not included in this repository.

## Dataset Description

The analysis uses a financial-risk assessment dataset containing
demographic, behavioural and financial variables.

The target variable represents three financial-risk categories:

- Low Risk
- Medium Risk
- High Risk
